Step-by-step tutorial

  2. Open CurseForge and sign in or continue as a guest.
  3. Click Import or create a new profile for Minecraft.

Method 2

Manual Installation

Install the modpack manually by dropping the mod files directly into your .minecraft/mods folder. Requires NeoForge to be installed separately.

Step-by-step tutorial

  1. Click the Download Modpack ZIP button above.
  2. Extract the downloaded ZIP archive to a temporary folder.
  3. Press Win + R to open the Run dialog.
  4. Type %appdata% and press Enter.
  5. Navigate to Roaming › .minecraft.
  6. Open (or create) the mods folder inside .minecraft.
  7. Copy all files from the extracted ZIP into the .minecraft folder.
  8. Download NeoForge using the button in the section below.
  9. Run the NeoForge installer and select Install client.
  10. Open the Minecraft Launcher, select the NeoForge profile, and launch!

⚙️ NeoForge Installer

NeoForge is the modloader required to run this modpack. Download the correct version for Minecraft 1.21.1, run the installer, and select Install client.
